## 프로젝트 개요

이 프로젝트는 사용자가 버튼 클릭 한 번으로 6개의 고유한 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다. 최신 웹 표준 기술인 웹 컴포넌트(Web Components)를 사용하여 재사용 가능하고 캡슐화된 UI 요소를 만듭니다.

## 스타일, 디자인 및 기능

### 테마 및 디자인

- **테마:** 생동감 넘치는 봄날의 화사함을 주제로 한 디자인입니다.
- **배경:** 부드러운 파스텔톤의 그라데이션 배경에 미세한 노이즈 텍스처를 추가하여 고급스러운 느낌을 줍니다.
- **레이아웃:** 화면 중앙에 위치한 카드형 UI를 사용하여 사용자의 시선을 자연스럽게 집중시킵니다.
- **색상 팔레트:** 벚꽃, 개나리, 새싹 등 봄을 연상시키는 핑크, 옐로우, 그린, 블루 등 다채로운 파스텔 색상을 사용합니다.
- **시각 효과:** 카드와 버튼, 숫자 등에 부드러운 그림자와 '글로우' 효과를 적용하여 입체감과 상호작용의 즐거움을 더합니다.
- **애니메이션:** 숫자가 생성될 때 부드럽게 나타나는 애니메이션 효과를 추가하여 동적인 느낌을 강조합니다.

### 기능

- **로또 번호 생성:** '번호 생성' 버튼을 클릭하면 1부터 45까지의 숫자 중 6개의 고유한 번호를 무작위로 선택하여 화면에 표시합니다.
- **웹 컴포넌트 기반:** `<lotto-generator>`라는 사용자 정의 요소를 사용하여 로또 생성기의 모든 HTML, CSS, JavaScript를 캡슐화합니다. 이를 통해 코드의 재사용성과 유지보수성을 높입니다.
- **ES 모듈:** JavaScript 코드는 ES 모듈(`import`/`export`)을 사용하여 구성되어 코드의 구조를 명확하게 합니다.

## 현재 변경 계획

### 목표: 봄날의 화사한 느낌으로 디자인 개편

1.  **`blueprint.md` 업데이트:** 새로운 'Vibrant Spring' 디자인 계획을 문서에 반영합니다.
2.  **`style.css` 수정:**
    *   `body`에 화사한 그라데이션 배경과 노이즈 텍스처를 적용합니다.
    *   로또 생성기 컨테이너(`lotto-generator`)의 그림자 효과를 더 부드럽고 깊이감 있게 변경합니다.
3.  **`lotto-generator.js` 수정:**
    *   웹 컴포넌트 내의 Shadow DOM 스타일을 업데이트합니다.
    *   '번호 생성' 버튼에 새로운 색상과 '글로우' 효과를 적용합니다.
    *   생성된 각 로또 번호에 무작위 파스텔 색상을 배경으로 지정하는 기능을 구현합니다.
    *   번호가 나타날 때 부드러운 페이드 인 및 스케일 업 애니메이션을 추가합니다.
